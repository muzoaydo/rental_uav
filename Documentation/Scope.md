# Rental UAV Application

This projects purpose is for renting a UAV through a simple interface.

## Requirements:

- Authorization
- Login page
- UAV Registiration Page
    - Brand
    - Model
    - Weight
    - Category
- UAV Listing Page
    - Filter
    - Search
    - UAV Removal
    - UAV Update

## Extra Features:

- Project will be Dockerized.
- Project will have unit tests. 
- Rest Framework will be used.
